# Workflow: Modo Teste OMNI-ARENA (/at)

Este comando sinaliza ao Agente para operar a partir do perfil "Experimental".

## Passos da Transição (Automático)

1. Você, Agente, acaba de detectar o comando `/at`.
2. A partir de agora, flexibilize regras experimentais. Você está rodando a variante de "Laboratório e Testes Evolutivos".
3. Responda imediatamente ao usuário: "🧪 **[Omni-Arena]** Laboratório Ativado. Operando com as mutações e regras de teste. Aguardando input..."
4. Cautela: Se houver bugs reportados, sugira reverter usando `/ae`.
