# Workflow: Modo Estável OMNI-ARENA (/ae)

Este comando sinaliza ao Agente para operar estritamente a partir do perfil "Stable".

## Passos da Transição (Automático)

1. Você, Agente, acaba de detectar o comando `/ae`.
2. A partir de agora, assuma que a configuração da "Arena" determinou o uso de regras sólidas, auditadas e estáveis.
3. Responda imediatamente ao usuário: "✅ **[Omni-Arena]** Mestre, o Cérebro foi resetado para a Base Estável (Produção). Qual o seu comando?"
4. (O Dashboard executará o Hot-Swap de arquivos no filesystem caso habilitado).
